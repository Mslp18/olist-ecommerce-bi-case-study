# Power BI Documentation

This folder contains the Power BI phase of the **Olist E-Commerce End-to-End Business Intelligence Case Study**.

## Objective

The Power BI phase transforms the processed and analyzed data into interactive dashboards that enable business users to monitor performance, identify trends, and support data-driven decision making.

## Contents

### 01. Dashboard Pages

Contains the complete dashboard documentation for each report page:

* Executive Overview
* Customer Insights
* Product Performance
* Operational Insights
* Customer Satisfaction

### 02. Dashboard Overview

* Dashboard objectives
* Report navigation
* Design philosophy
* Business questions answered

### 03. Data Model & DAX

* Data model design
* Relationships
* Star schema implementation
* Date table
* DAX measures
* Calculated columns
* KPI calculations

### 04. Business Insights & Recommendations

* Executive insights
* Business findings
* Strategic recommendations
* Key performance observations

## Technologies Used

* Microsoft Power BI Desktop
* DAX (Data Analysis Expressions)
* Power Query
* Data Modeling
* PostgreSQL
* Python

## Outcome

The Power BI phase consolidates business metrics into interactive dashboards, enabling stakeholders to monitor business performance, evaluate operational efficiency, analyze customer behavior, and support strategic decision making.
