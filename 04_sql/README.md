# SQL Documentation

This folder contains the SQL phase of the **Olist E-Commerce End-to-End Business Intelligence Case Study**.

## Objective

The SQL phase transforms the cleaned dataset into actionable business insights through exploratory data analysis, relational queries, KPI development, business analysis, and advanced SQL techniques.

Each module includes:

* SQL query scripts (`.sql`)
* Supporting documentation (`.pdf`) explaining the analytical approach, methodology, and business findings.

## Folder Structure

### 00. Database Fixes

* Database corrections and validation before analysis.

### 01. Exploratory Data Analysis (EDA)

* Customer Analysis
* Order Analysis
* Payment Analysis
* Review Analysis
* Product Analysis

### 02. Join Analysis

* Multi-table joins
* Relationship validation
* Cross-table business analysis

### 03. KPI Development

* Business KPI calculations
* Revenue metrics
* Customer metrics
* Operational performance indicators

### 04. Business Analysis

* Business-focused SQL analysis
* Performance evaluation
* Decision-support queries

### 05. Hypothesis Testing

* SQL-based validation of business hypotheses
* Data-driven verification of assumptions

### 06. Advanced SQL

* Common Table Expressions (CTEs)
* Window Functions
* Subqueries
* Ranking and analytical SQL techniques

### 07. SQL Insights Summary

* Consolidated analytical findings
* Key business insights
* Recommendations for decision making

## Technologies Used

* PostgreSQL

* SQL

## Recommended Reading Order

To understand the SQL workflow from data exploration to business insights, review the modules in the following order:

1. Database Fixes
2. Exploratory Data Analysis (EDA)
3. Join Analysis
4. KPI Development
5. Business Analysis
6. Hypothesis Testing
7. Advanced SQL
8. SQL Insights Summary

## Outcome

The SQL phase converts prepared data into meaningful business insights through structured querying, KPI development, advanced analytics, and hypothesis validation. These outputs form the analytical foundation for the Power BI dashboards and final business recommendations.
